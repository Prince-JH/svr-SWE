# Description

영화 재개봉 요청 플랫폼을 위한 백엔드 서버입니다.


# Getting Started

필요 패키지 설치  
**pip install -r requirement.txt**

서버 구동
python3 manage.py runserver

swagger 주소 (8000포트 기준)
http://127.0.0.1:8000/swe/swagger/
